# IBM Legal Document Classification Tool

## 📌 Overview

This project classifies legal document summaries into different categories using Natural Language Processing (NLP) and Machine Learning.

The model predicts the following document types:

- Contract
- Service Agreement
- Non-Disclosure Agreement (NDA)
- Data Sharing Agreement

---

## 🚀 Features

- Upload legal document summaries
- Text preprocessing using TF-IDF
- Document classification using Multinomial Naive Bayes
- Predict document type
- Evaluate model performance

---

## 🛠 Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Google Colab

---

## 📂 Dataset

The dataset contains legal document summaries and their corresponding document categories.

---

## 🤖 Machine Learning Workflow

1. Data Collection
2. Data Preprocessing
3. TF-IDF Vectorization
4. Model Training (Multinomial Naive Bayes)
5. Model Evaluation
6. Document Prediction

---

## 📊 Model

- TF-IDF Vectorizer
- Multinomial Naive Bayes Classifier
